# מסמך אפיון מערכת
## מערכת ניהול אימונים מקצועית למאמני טניס
### PWA - Progressive Web Application

**גרסה 1.0 | ינואר 2026**

---

## 1. סקירה כללית

### 1.1 מטרת המערכת
המערכת נועדה לספק פתרון דיגיטלי מקיף לניהול אימוני טניס במרכזי טניס ברחבי הארץ. המערכת מאפשרת למאמנים לתכנן אימונים, לעקוב אחר התקדמות שחקנים, לנהל קבוצות, ולתעד מטרות ותרגילים.

### 1.2 קהל יעד
- מאמני טניס מקצועיים העובדים במרכזי טניס
- מנהלי מרכזי טניס
- מנהלים מקצועיים (Supervisors)

### 1.3 יתרונות PWA
- עבודה אופליין - המערכת תעבוד גם ללא חיבור לאינטרנט
- התקנה על מכשיר נייד - חוויית אפליקציה מקומית
- עדכונים אוטומטיים - ללא צורך בהורדה מחנות אפליקציות
- חוצה פלטפורמות - עובד על iOS, Android, ומחשב

## 2. תפקידים והרשאות

### 2.1 מנהל מקצועי (Supervisor)
- גישה לכל המרכזים והמאמנים
- צפייה בדשבורד מרכזי עם סטטיסטיקות
- הגדרת מטרות ברמת הארגון
- ניהול ערכים ותרגילים גלובליים
- מעקב תוכניות חודשיות

### 2.2 מנהל מרכז
- ניהול המרכז הספציפי שלו
- צפייה בסטטיסטיקות המרכז
- ניהול קבוצות ומאמנים במרכז
- לוח אירועים מרכזי
- מעקב מטרות חודשיות

### 2.3 מאמן
- ניהול הקבוצות שלו
- תכנון וביצוע אימונים
- הגדרת תרגילים
- מעקב התקדמות שחקנים

## 5. מסכים ותצוגות

### 5.1 דשבורד מנהל מקצועי
- גרפים של מטרות שהושגו לפי חודש
- מעקב תוכניות חודשיות - כמה הוגשו ומתי
- לוח שנה מרכזי עם כל האירועים
- סטטוס מרכזים - פעילים, צריכים תשומת לב

### 5.2 דשבורד מנהל מרכז
- לוח שנה חודשי עם כל האירועים
- מעקב בניית תוכניות חודשיות
- סטטוסים בזמן אמת - כמה תוכניות נבנו
- רשימת ערכים חודשיים
- מטרת החודש

### 5.3 הגדרת מטרות
- מטרות לפי חודש
- מטרות לפי קבוצה/מאמן
- סימון מטרות שהושגו
- היסטוריית מטרות

### 5.4 רשימת מנהלי מרכזים
- תצוגת גריד של כל מנהלי המרכזים
- גישה מהירה לכל מרכז
- חיפוש וסינון

### 5.5 מסך תרגילי טניס
- רשימת כל התרגילים במערכת
- סינון לפי: מצב משחק, רמה, נושא
- תצוגת כרטיסיות עם הסבר וסרטון
- בחירה מהמאגר להוספה לאימון

### 5.6 טופס בקשה לתרגיל חדש
- שם מלא של הכותב
- שם התרגיל
- הסבר מפורט
- רמת משחק (מתחילים, מתקדמים, מתקדמים מאוד)
- מצב משחק (עד 3 בחירות)
- נושא התרגיל (עד 4 בחירות)
- העלאת סרטון הדגמה (עד 100MB)

### 5.7 ניהול בקשות לתרגילים
- תצוגת טבלה של כל הבקשות
- סטטוס: ממתין, מאושר, לא מאושר
- אפשרות אישור והוספה למאגר

### 5.8 הגדרת ערכים
- ערכים חודשיים לכל המרכז
- רשימת ערכים עם עדיפות
- הוספה ועריכה של ערכים

### 5.9 מסך תכנון אימון
- כותרת: אימון הכנה לתחרות ארצית (או לפי סוג התקופה)
- בחירת קבוצה ותאריך
- הגדרת סוג תקופה, מצב משחק, מרכיב משחק
- בחירת נושאי אימון (Multi-select)
- שדה פירוט חופשי
- טבלת תרגילים עם:
  - בחירת תרגיל מהמאגר (Autocomplete)
  - זמן תרגיל
  - רמה, מצב משחק, נושא - נטענים אוטומטית
  - סרטון הדגמה
  - הסבר התרגיל

### 5.10 מסך כרטיס קבוצה/שחקן
- פרטי הקבוצה/שחקן
- שיוך למרכז ומאמן
- קטגוריית גיל ורמה
- שנתונים
- התראה: נא למלא את כל הנתונים - מרכז, מאמן, גיל ורמה, שנתונים

## 6. תהליכים עיקריים

### 6.1 תהליך תכנון אימון
1. מאמן בוחר קבוצה ותאריך

4. מנהל מקצועי מקבל התראה
5. מנהל מאשר/דוחה
6. אם אושר - התרגיל נוסף למאגר

### 6.3 תהליך מעקב מטרות חודשיות
1. מנהל מקצועי מגדיר מטרות לחודש
2. מטרות מופצות למרכזים הרלוונטיים
3. מנהלי מרכזים מסמנים השגת מטרות
4. דשבורד מרכזי מציג התקדמות

## 7. דרישות טכניות

### 7.1 PWA Requirements
- Service Worker לעבודה אופליין
- Manifest file להתקנה
- Responsive Design לכל גדלי מסך
- Push Notifications להתראות
- Background Sync לסנכרון נתונים

### 7.2 Frontend Stack (מומלץ)
- React או Vue.js
- TypeScript
- Tailwind CSS או Material UI
- PWA plugins (Workbox)

### 7.3 Firebase Backend
- Firebase Authentication - התחברות באמצעות מייל ארגוני בלבד
- Cloud Firestore - מסד נתונים NoSQL בזמן אמת
- Firebase Storage - אחסון קבצים וסרטונים
- Firebase Cloud Functions - לוגיקה בצד שרת
- Firebase Hosting - אירוח האפליקציה

### 7.4 אימות והרשאות
- **התחברות באמצעות Email/Password בלבד (ללא Google Sign-In)**
- **הגבלת רישום למיילים ארגוניים בלבד (דומיין מאושר)**
- Firebase Security Rules להגנה על נתונים
- Role-based access control לפי תפקיד המשתמש

### 7.5 אחסון ומדיה
- Firebase Storage לאחסון סרטונים עד 100MB
- Firebase CDN להעלאת תמונות ולוגואים
- Offline persistence עם Firestore
- Caching לביצועים

## 9. אימות ואינטגרציות

### 9.1 מערכת התחברות
- **התחברות באמצעות מייל ארגוני בלבד**
- **אין אפשרות להתחבר דרך Google או ספקים חיצוניים**
- רק מיילים מדומיין מאושר יכולים להירשם
- שכחתי סיסמה - איפוס דרך מייל
- Session management עם Firebase Auth

### 9.2 Firebase Services
- Firestore - מסד נתונים ראשי עם סנכרון real-time
- Firebase Storage - אחסון סרטונים ותמונות
- Cloud Functions - לוגיקה עסקית (התראות, ולידציות)
- Firebase Analytics - מעקב שימוש (אופציונלי)

### 9.3 העלאת סרטונים
- העלאה ישירה ל-Firebase Storage
- תמיכה ב-YouTube embeds כקישור חיצוני
- לימיט: 100MB לקובץ
- סוגי קבצים: MP4, MOV, AVI

### 9.4 התראות
- Firebase Cloud Messaging (FCM) למובייל ודסקטופ
- התראות על אימונים חדשים, מטרות, בקשות לתרגילים
- Email notifications דרך Firebase Extensions (אופציונלי)

## 10. סיכום

מסמך זה מתאר את האפיון המלא למערכת ניהול אימוני טניס מבוססת PWA ו-Firebase. המערכת מיועדת לספק חוויית משתמש מקצועית עם יכולות אופליין, ממשק מותאם מובייל, אבטחה באמצעות מייל ארגוני, וניהול נתונים בזמן אמת.

### השלבים הבאים:
1. אישור האפיון
2. עיצוב UI/UX מפורט
3. בחירת טכנולוגיות סופית
4. פיתוח MVP
5. בדיקות ושיפורים
6. השקה
